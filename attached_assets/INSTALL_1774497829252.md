# Alpha Lens — Update Installation Guide
## What this update adds: Daily Briefing + Live Trading

---

## WHAT'S IN THIS UPDATE

**E6 — Proactive Recommendations Engine**
The AI agent now scans global markets every 30 minutes and sends you
"Hey! Check out these potential trades!" alerts automatically.
- Searches the internet for live global events
- Cross-references events against scored assets
- Produces ranked TRADE CALLS and WATCHES with historical context
- Accessible at: /briefing in your app

**E7 — Live Trading Engines**
Connect real trading accounts and execute directly from AI recommendations.
- Polymarket  (Dominica / Philippines — not US)
- Kalshi      (US, Philippines, Dominica, 140+ countries)
- Alpaca      (stocks/ETFs — available internationally)

---

## STEP 1 — Upload this file to Replit

1. In your Replit project, click the three-dot menu next to Files
2. Choose "Upload file"
3. Upload: install_updates.sh
4. The file will appear in your root directory

---

## STEP 2 — Run the installer

In the Replit Shell tab, type:
```
bash install_updates.sh
```

This takes about 10 seconds and creates all new files automatically.
It does NOT delete or break any existing files.

---

## STEP 3 — Add new secrets to Replit

Click the padlock icon (Secrets) in the left sidebar.
Add these — only the ones relevant to your platforms:

**For Kalshi (Philippines + US + Dominica):**
```
KALSHI_EMAIL         your-email@example.com
KALSHI_PASSWORD      your-kalshi-password
```

**For Polymarket (Philippines + Dominica — not US):**
```
POLYMARKET_API_KEY      from polymarket.com/builders
POLYMARKET_SECRET       from polymarket.com/builders
POLYMARKET_PASSPHRASE   from polymarket.com/builders
POLYGON_WALLET_ADDRESS  your 0x wallet address
```

**For Alpaca (stocks/ETFs — available everywhere):**
```
ALPACA_API_KEY      from alpaca.markets
ALPACA_SECRET_KEY   from alpaca.markets
ALPACA_BASE_URL     https://paper-api.alpaca.markets   (paper first!)
```

**Risk controls (add all of these):**
```
EXECUTION_MODE       manual
REQUIRE_APPROVAL     true
MIN_EDGE_TO_EXECUTE  5
MIN_CONFIDENCE       65
MAX_POSITION_PCT     0.05
MAX_DAILY_TRADES     5
DAILY_LOSS_LIMIT_PCT 0.05
```

---

## STEP 4 — Add new database tables

1. Go to supabase.com → your project
2. Click "SQL Editor" in the left sidebar
3. Open the file: scripts/add_new_tables.sql
   (it was created by the installer in step 2)
4. Copy the entire contents
5. Paste into the Supabase SQL Editor
6. Click "Run"

You should see: "Success. No rows returned"

---

## STEP 5 — Restart the app

Click the green Run button in Replit.
You should see in the console:
```
✓ Alpha Lens v3.0 started
  Jobs: markets(15min) · research(60min) · recommendations(30min)
```

---

## STEP 6 — Verify everything works

Test these URLs (replace with your Replit URL):

**Check platform connections:**
```
GET https://your-repl.replit.app/api/trading/accounts
```
Should show: configured / not_configured for each platform

**Trigger first AI scan:**
```
POST https://your-repl.replit.app/api/recommendations/scan
```
Wait 60 seconds, then:

**Get your first briefing:**
```
GET https://your-repl.replit.app/api/recommendations/briefing
```
Should return trade calls and watches

**Visit /briefing in your app** to see the full daily intelligence briefing UI

## QUESTIONS?

If the installer fails, check:
1. You're in the root directory (backend/main.py should exist)
2. You have write permissions (you do on paid Replit)
3. The Shell tab is showing, not the Console tab

The Console tab shows app output. The Shell tab lets you type commands.
