Investigate the AI Coach API call and reduce latency so responses arrive in under 10 seconds. Apply all of the following that are applicable:

- Enable streaming responses on the coach endpoint so the UI begins rendering tokens as they arrive rather than waiting for the full response. Update the frontend to consume a streaming response and progressively display text.

- For Global Events severity labels, replace the existing labels with impact-framing labels: "Impact: Low", "Impact: Medium", "Impact: High". Update the badge/chip text and any underlying value-to-label mappings.
- For Watch and Avoid calls severity labels, use the same impact-framing labels as Global Events ("Impact: Low / Medium / High") since these are informational, not actionable conviction calls.
- For Trade Calls, retain the current conviction-based labels ("Trade: Medium / High / Critical") but confirm the label prefix makes the distinction clear.
- Add the same explanatory tooltip that currently exists on Trade Call severity badges to Watch, Avoid, and Global Events severity badges. The tooltip text should reflect the appropriate framing: for Global Events and Watch/Avoid, the tooltip should explain market impact level, not trade conviction.
- Update any legend or key on the page if one exists.

**4. Prevent text wrapping in the Asset column of the Open Positions table (P3-13) — /portfolio**

The Win Rate figure in the Track Record section lacks context, making it feel misleadingly low.
