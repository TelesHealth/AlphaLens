Add BEA (Bureau of Economic Analysis) GDP data to
artifacts/api-server/src/services/macro-data.ts

BEA API base URL:
https://apps.bea.gov/api/data

API key env var: process.env.BEA_API_KEY

STEP 1 — Add fetchGDP() function

GET request to BEA API:
https://apps.bea.gov/api/data?
  UserID={BEA_API_KEY}
  &method=GetData
  &DataSetName=NIPA
  &TableName=T10101
  &Frequency=Q
  &Year=LAST5
  &ResultFormat=JSON

Parse response:
  Find the "Gross domestic product" line item
  in BEAAPIResponse.BEAAPI.Results.Data
  Filter where LineDescription === 
    "Gross domestic product"
  Sort by TimePeriod descending
  Take two most recent quarters

  current = parseFloat(data[0].DataValue
    .replace(/,/g, ''))
  previous = parseFloat(data[1].DataValue
    .replace(/,/g, ''))
  change = parseFloat(
    (((current - previous) / previous) * 100)
    .toFixed(2)
  )
  period = data[0].TimePeriod (e.g. "2025Q4")

Return:
{
  current,
  previous,
  change,
  period,
  source: "BEA"
}

On any error: return null. Never throw.
Log: "BEA GDP: {current} (QoQ change: {change}%)"
On failure: "BEA GDP: fetch failed"

Add 24-hour cache — GDP is quarterly data.
Short TTL (5 min) on failures.
