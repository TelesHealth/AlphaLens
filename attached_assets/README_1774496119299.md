# Alpha Lens — AI-Powered Global Investment Intelligence

Version 3.0 | FastAPI + Next.js | Replit-optimized

---

## What This Is

Alpha Lens is a full-stack investment intelligence platform that:
- Scores **Polymarket prediction markets, crypto, stocks, and commodities** using Claude AI
- Provides a **live probability coach** that updates like WSOP poker odds as signals arrive
- Tracks **global geopolitical signals** across BRICS+, EU, APEC, Latin America, Africa, MENA
- Includes **paper trading** with $10,000 imaginary balance and AI post-trade debriefs
- Covers **adversarial powers** (China, Russia) with shadow-economy signal protocols

## Quickstart on Replit (10 minutes)

### Step 1 — Create external services (free tiers)

**Supabase (database):**
1. Go to https://supabase.com → New project
2. Copy: Project URL, anon key, service role key

**Upstash Redis (cache):**
1. Go to https://upstash.com → Create database → REST API
2. Copy: REST URL, REST token

**Anthropic (AI engine):**
1. Go to https://console.anthropic.com → API Keys → Create key

### Step 2 — Add secrets to Replit

In your Repl, click the **padlock icon** (Secrets) in the left sidebar.
Add these secrets one by one:

| Secret Name | Where to get it |
|---|---|
| `ANTHROPIC_API_KEY` | console.anthropic.com |
| `SUPABASE_URL` | Supabase → Project Settings → API |
| `SUPABASE_ANON_KEY` | Supabase → Project Settings → API |
| `SUPABASE_SERVICE_KEY` | Supabase → Project Settings → API |
| `UPSTASH_REDIS_URL` | Upstash → Database → REST URL |
| `UPSTASH_REDIS_TOKEN` | Upstash → Database → REST Token |
| `COINGECKO_API_KEY` | coingecko.com/api (optional) |

### Step 3 — Create database tables

1. Go to your Supabase dashboard
2. Click **SQL Editor** in the left sidebar
3. Paste the entire contents of `scripts/schema.sql`
4. Click **Run**

### Step 4 — Install frontend dependencies

In the Replit **Shell** tab:
```bash
cd frontend && npm install && cd ..
```

### Step 5 — Click Run

Hit the green **Run** button. The startup script will:
1. Install Python dependencies (~2 min first time)
2. Build the Next.js frontend
3. Start FastAPI on port 8000

### Step 6 — Seed initial markets

In the Shell:
```bash
python scripts/seed_markets.py
```

### Step 7 — Test the AI pipeline

```bash
python scripts/test_signal.py
```

This runs one complete cycle: fetch a Polymarket market → research with Claude → score → print result.

## API Endpoints

| Method | Path | Description |
|---|---|---|
| GET | `/api/markets` | List all markets (filterable) |
| GET | `/api/markets/{id}` | Market detail + evidence |
| POST | `/api/markets/{id}/score` | Trigger AI research + scoring |
| POST | `/api/markets/refresh` | Refresh all market data |
| GET | `/api/portfolio/` | Paper trading portfolio |
| POST | `/api/portfolio/trade` | Open paper trade |
| POST | `/api/portfolio/close` | Close paper trade |
| GET | `/api/signals/{asset_id}` | Evidence records for asset |
| POST | `/api/coach/analyze` | Get AI coach note |
| GET | `/api/health` | Health check |
| WS | `/ws/signals/{asset_id}` | Live signal WebSocket |

## Background Jobs (APScheduler)

Runs inside FastAPI — no separate worker needed on Replit:

- **Every 15 minutes:** `fetch_all_markets` — refreshes prices from Polymarket, CoinGecko, Yahoo Finance
- **Every hour:** `run_research_agent` — re-scores top 10 markets with Claude AI

---

## Replit-Specific Notes

- Uses **APScheduler** instead of Celery (no separate worker process needed)
- Uses **Upstash Redis** (serverless, connects over HTTP — no persistent connection)
- Next.js built with `output: 'export'` — static files served by FastAPI
- All secrets stored in **Replit Secrets**, never in code
- **Always-on:** Use Replit Deployments (included in paid plans) to keep the app running
